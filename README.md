# Sleep Analysis Project

This project analyzes sleep patterns across mammalian species and correlates sleep with predation.

## Objectives
- Clean raw dataset
- Compute descriptive statistics
- Visualize sleep of mammals
- Find a correlation between hours slept and predation.

## Tools Used
- R
- ggplot2
- dplyr

## Key Finding
Prey vulnerability appears negatively associated with total sleep duration. Those mammals that are most likely to be prey, sleep less.
